# MAL-Money-Analytics-Ledger-
Built **MAL (Money Analytics Ledger)**, a responsive personal finance dashboard with transaction management, budget tracking, goal monitoring, interactive Chart.js visualizations, report export, and an AI assistant interface using HTML, CSS, JavaScript, and LocalStorage.
